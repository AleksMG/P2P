Technical Implementation

This implementation uses HMAC-SHA512 for cryptographically secure S-Box generation.

Algorithm Steps:

S-Box generation using HMAC-SHA512 based on the key
Affine transformation: y = (a * x + b) mod 26
Text splitting into bigrams (2 characters)
CBC mode with affine transformation for block chaining
Each bigram encrypted with a random homophone from S-Box
Entropy testing for cryptographic strength analysis
