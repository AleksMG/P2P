Как это работает:

Этот инструмент реализует омофонический шифр для английского алфавита с использованием квадрата 26x26. Каждая ячейка в квадрате соответствует биграмме (паре букв) и содержит несколько омофонов из набора Base64.

Всего существует 676 возможных биграмм (26x26) и 4096 уникальных омофонов (64x64), что позволяет распределить по 6 омофонов на большинство ячеек, а на 40 ячеек - по 7.

Ключевая особенность: При шифровании для каждой биграммы случайно выбирается один из нескольких омофонов, поэтому один и тот же текст с одним ключом будет давать разный шифротекст!

Для использования:

  1. Введите ключ (любую фразу)
  2. Нажмите "Сгенерировать таблицу"
  3. Введите текст для шифрования/расшифровки
  4. Нажмите соответствующую кнопку

Один и тот же ключ необходим как для шифрования, так и для расшифровки.
